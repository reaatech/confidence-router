# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in `confidence-router`, please report it responsibly.

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Instead, report via:

- **GitHub Security Advisories**: [Open a private advisory](https://github.com/reaatech/confidence-router/security/advisories/new)
- **Email**: security@reaatech.dev

We aim to respond to security reports within 48 hours and will work with you to verify, address, and disclose the issue appropriately.
